# Fake-News-Detection

## Description
This project is a Machine Learning based web application that detects whether a news article is Fake or Real using NLP techniques.

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas

## How to Run

1. Download the dataset (Fake.csv & True.csv)
2. Place them in the same folder as app.py
3. Run:
   streamlit run app.py

## Features
- Detect Fake or Real News
- Shows Confidence Score
- Simple Web Interface
